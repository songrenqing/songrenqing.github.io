    博客上增加一些新的功能

    srq 工具 NexT Bug
    创建时间:2020-03-08 10:37
    字数:268 阅读:
    评论:
    1. 新的功能
      1. 百度统计
      2. busuanzi显示访问人数
    2. Bug
      1. 边栏头像下面的日志链接错误

    新的功能

    百度统计

    利用百度统计，可以查看网站的流量数据、来源分析等。但是一般博客的网站流量都不会太高，除非博主是业内比较有名的技术大牛，自带流量。

    1. 先注册一个账号注册页面
    2. 管理-代码管理-代码获取，复制代码ID
    3. ID复制到config.yml文件中的baidu_analytics后面，重新发布，大概20分钟左右就可以看到网站的访问数据。

    busuanzi显示访问人数

    找到busuanzi-counter的源文件

    /themes/next/layout/_third-party/analytics/busuanzi-counter.swig

    将原代码

    <script async src="https://dn-lbstatics.qbox.me/busuanzi/2.3/busuanzi.pure.mini.js"></script>

    修改为

    <script async src="https://busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>

    然后将_config.yml中的busuanzi_count的enable改为true

    Bug

    边栏头像下面的日志链接错误

    找到边栏的配置文件

    /themes/next/layout/_macro/sidebar.swig

    将原代码

    <a href="{{ url_for(theme.menu.archives).split('||')[0] | trim }}">

    修改为

    <a href="{{ url_for(config.archive_dir) | trim }}">
